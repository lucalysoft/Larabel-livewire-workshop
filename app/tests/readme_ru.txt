Как настроить консоль PHP (Windows)
-----------------------------------
1. К переменной окружения PATH добавляем полный путь к php.exe.

Особенности консоли PHP в Denwer
--------------------------------
1. Отредактируйте php.ini так, чтобы все пути были абсолютными
   от корня виртуального диска Denwer.
2. Иногда требуются некоторые dll. Их надо положить в папку PHP.

Как установить PHPUnit
----------------------
1. Ставим PEAR: сохраняем http://pear.php.net/go-pear как файл go-pear.php в папке, куда будем ставить PEAR.
2. Запускаем из консоли php go-pear.php. Отвечаем на задаваемые вопросы.
3. Добавляем к переменной окружения PATH полный путь к установленному PEAR. 
3. Ставим PHPUnit. В консоли:
   pear channel-discover pear.phpunit.de
   pear install phpunit/PHPUnit

Подготовка БД
-------------
Для подготовки БД необходимо выполнить SQL из app/tests/sql.

Как запустить тесты
-------------------
В консоли запускаем app/tests/rununit.bat или phpunit --verbose unit.
